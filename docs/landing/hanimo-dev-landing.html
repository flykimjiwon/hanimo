Features

Built for developers, by developers.

Everything you need from a modern coding agent — wrapped in a single binary you can drop anywhere.

01

14+ LLM Providers

Ollama, OpenAI, Anthropic, Google, Novita, OpenRouter, DeepSeek, Groq, Together, Fireworks, Mistral, vLLM, LM Studio, or any custom OpenAI-compatible endpoint.

02

Single Binary · ~20MB

One statically compiled Go executable. No Node, no Python, no venvs. Download, chmod, run. Ships for every major platform.

03

Hash-Anchored Editing

Edits are anchored to content hashes, so the agent never silently overwrites drifted files. Safe, surgical diffs by default.

04

Smart Compaction

Automatic context compaction that preserves tool calls, file anchors, and decisions — not just the last N messages.

05

SQLite Sessions

Every session persisted locally in SQLite. Resume, fork, export. Your history never leaves your machine.

06

MCP Support

First-class Model Context Protocol support — plug in any MCP server for filesystems, databases, APIs, or custom tools.

07

Autonomous Mode

Hand off full tasks with guardrails. Loop-until-done with test-driven verification and bounded retry budgets.

08

Command Palette

⌘K anywhere to jump modes, switch providers, or fire commands. Shortcut-first UX you actually want to live in.

09

5 Built-in Themes

Mocha, Latte, Dracula, Gruvbox, Tokyo Night — gorgeous out of the box. Or fork your own palette in minutes.

Quick Start

Up and running in 30 seconds.

Download the binary for your platform, make it executable, and pair it with any LLM you trust — from a local Ollama model to Anthropic's Claude.

  1. 1
    Grab the single static binary from GitHub releases.
  2. 2
    Pick a provider — local (Ollama) or cloud (OpenAI, Anthropic, Google, …).
  3. 3
    Drop into your repo and start pairing. That's the whole setup.
~/bin — install.sh 
# Download
curl -L -o hanimo \
  https://github.com/flykimjiwon/hanimo/releases/latest/download/hanimo-darwin-arm64
chmod +x hanimo

# Run with Ollama (no API key needed)
ollama pull qwen3:8b
./hanimo

# Or use any provider
./hanimo -p openai     -m gpt-4o
./hanimo -p anthropic  -m claude-sonnet-4
./hanimo -p novita     -m openai/gpt-oss-120b
